# Stack Expression Evaluator

An interactive visualizer that converts mathematical expressions from **Infix to Postfix** using the Shunting-Yard algorithm and evaluates them step by step using a stack.

## Live Demo
🔗 [https://expression-evaluater.vercel.app/)

## Features
- Step-by-step stack animation
- Infix to Postfix conversion (Shunting-Yard algorithm)
- Postfix evaluation using stack
- Operator precedence visualization
- Color-coded tokens (numbers, operators, parentheses)

## DSA Concepts Used
- Stack (push & pop)
- Shunting-Yard Algorithm
- Postfix Expression Evaluation
- Operator Precedence

## Tech Stack
- HTML, CSS, JavaScript (no libraries)
